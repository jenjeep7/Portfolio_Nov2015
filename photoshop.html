Photoshop

Photoshop is a project based class that introduced web graphic fundamentsls using the latest, Photoshop CC. Projects included creating and enhancing digital images using masks, layers and filters. The basics of vector images was also taught using Adobe Illustrator. This course covered designing for print as well as optimizing images for web use.

This movie ad was created using techniques such as incorporating vector objects, using smart layers, scaling, alignment and managing layout.

This advertising project was created using color replacement, adjustment layers, shadows/highlights, warping and lighting effects to make an added element appear natural.

In this project tools such as blur galleries, content-aware tools, text layer types and 3D were used to create a visually appealing promotional post card.

One of the most important skills in Photoshop is being able to adjust and repair images. This is an example of an image that was corrected using the various tools available in Photoshop.

This drawing was created using brushes and layers over a vector image to give the image color and more realistic qualities.

The photo above is the result of the introduction to resizing, resampling, using vector paths, shape layers and applying non-destructive styes.

Because this class is a requirement for the Web Development program, this project was done to how to design for web pages in Photoshop

This calendar cover is the result of learning how to manage missing fonts, creating vector shapes, creating a complex mask and creating spot channels for print.
